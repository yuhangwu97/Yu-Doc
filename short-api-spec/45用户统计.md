# 统计模块新增接口文档

本文档描述了新增的活跃用户统计、VIP充值统计以及系统用户统计接口。

## 1. 活跃用户统计汇总

获取按需统计的活跃用户数据。

**接口地址:** `/api/user/active/statistics/summary`
**请求方式:** `GET`

### 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
| --- | --- | --- | --- | --- |
| type | String | 否 | day | 统计维度: `day` (日), `week` (周), `month` (月), `year` (年) |
| startTime | String | 否 | - | 开始时间 (格式: yyyy-MM-dd) |
| endTime | String | 否 | - | 结束时间 (格式: yyyy-MM-dd) |

### 响应示例

```json
{
  "success": true,
  "message": "操作成功！",
  "code": 200,
  "result": [
    {
      "time": "2023-10-01",
      "totalRevenue": 150,  // 注意：此处复用了 totalRevenue 字段表示“活跃用户数”
      ...
    }
  ],
  "timestamp": 1696147200000
}
```
