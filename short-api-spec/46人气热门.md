
## 1. 高分优质 (High Score & High Quality)
**描述**: 获取按**消费订单量**（支付订单数）倒序排列的短剧列表。

- **URL**: `/api/order/orderPayRecharge/getDramaOrderStats`
- **Method**: `GET`
- **Authentication**: 需要 (Token in Header)

### 请求参数 (Request Parameters)
*无必填参数。*
*(注：需求中提到的 `column`, `order`, `pageNo`, `pageSize` 参数目前被该接口**忽略**，因为该接口返回的是按总订单数排序的统计聚合列表。)*

### 响应示例 (Response Example)
```json
{
  "success": true,
  "message": "操作成功",
  "code": 200,
  "result": [
    {
      "filmDramaId": "1234567890",
      "dramaName": "霸道总裁的秘密",
      "dramaPoster": "http://example.com/poster.jpg",
      "dramaDescribe": "一个关于爱与复仇的故事...",
      "classifyName": "都市",
      "producerName": "星空影视",
      "orderCount": 1500,
      "totalAmount": null
    },
    {
      "filmDramaId": "0987654321",
      "dramaName": "王者归来",
      "dramaPoster": "http://example.com/poster2.jpg",
      "dramaDescribe": "战神回归，发现女儿住狗窝...",
      "classifyName": "战神",
      "producerName": "火焰传媒",
      "orderCount": 1200,
      "totalAmount": null
    }
  ],
  "timestamp": 1700000000000
}
```

### 响应字段说明 (`DramaOrderStatVo`)
| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| `filmDramaId` | String | 剧目 ID (唯一标识) |
| `dramaName` | String | 剧目名称 |
| `dramaPoster` | String | 剧目封面 URL |
| `dramaDescribe` | String | 剧目介绍/简介 |
| `classifyName` | String | 分类名称 |
| `producerName` | String | 制作方名称 |
| `orderCount` | Integer | 有效支付订单总数 |

## 2. 经典回顾 (Classic Review)
**描述**: 获取按**历史总播放量**（真实+虚拟）倒序排列的短剧列表。
*包含状态为“上架” (Status=1) 的短剧。*

- **URL**: `/api/appApi/classicList`
- **Method**: `GET`
- **Authentication**: 需要 (Token in Header)

### 请求参数 (`AppDramaVo`)
该接口支持通过查询参数进行过滤。

| 参数名 | 类型 | 必填 | 描述 |
| :--- | :--- | :--- | :--- |
| `dramaName` | String | 否 | 搜索剧目名称 (模糊匹配) |
| `tenantId` | Integer | 否 | 按租户 ID 过滤 |
| `sysOrgCode` | String | 否 | 按部门编码过滤 |

*注：目前未强制分页，直接返回按热度排序的完整列表。*

### 响应示例 (Response Example)
```json
{
  "success": true,
  "message": "操作成功",
  "code": 200,
  "result": [
    {
      "id": "1856723...",
      "dramaId": "1001",
      "dramaName": "悟空传",
      "dramaPoster": "http://example.com/poster.jpg",
      "dramaDescribe": "五百年前，石猴出世...",
      "classifyName": "动作",
      "producerName": "西游影业",
      "totalPlay": "120.5万",
      "totalEpisodes": 80,
      "videoUrl": "http://example.com/episode1.mp4",
      "dramaSignList": ["1"],
      ...
    },
    ...
  ],
  "timestamp": 1700000000000
}
```

### 关键响应字段说明 (`AppDramaVo`)
| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| `dramaId` | String | 剧目业务 ID |
| `dramaName` | String | 剧目名称 |
| `dramaPoster` | String | 封面图片 URL |
| `dramaDescribe` | String | 剧目介绍 |
| `classifyName` | String | 分类名称 |
| `producerName` | String | 制作方名称 |
| `videoUrl` | String | 第一集播放地址 (用于预览) |
| `totalEpisodes` | Integer | 总集数 |
