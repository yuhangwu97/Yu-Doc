# 团队数据API集成文档

---

## 1. 获取团队统计数据

### 接口说明
获取团队人员数量和总利润数据，用于展示团队概览统计卡片。

### 接口地址
```
GET /api/appApi/appDivideApi/getTeamData
```

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `memberId` | String | 是 | 会员ID |

### 返回数据

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `totalExtendNumber` | Integer | 团队总人数（一级+二级推广成员） |
| `totalProfit` | BigDecimal | 总利润（金币分佣+充值分佣） |
| `todayGold` | BigDecimal | 今日金币分佣 |
| `todayGiveaway` | BigDecimal | 今日赠送分佣 |
| `todayNewArrivals` | Integer | 今日新增成员数 |

---

## 2. 获取团队成员列表

### 接口说明
获取团队成员详细列表，包含成员名称、绑定时间、收益等数据。

### 接口地址
```
GET /api/appApi/appDivideApi/getTeamDataList
```

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `memberId` | String | 是 | 会员ID（查询该会员的团队成员） |
| `divideType` | Integer | 否 | 分销分类（1-一级，2-二级，不传则返回全部） |

### 返回数据

返回 `DivideRecordByTeamVo[]` 数组，每个对象包含：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `memberId` | String | 会员ID |
| `memberName` | String | 会员昵称 |
| `bindTime` | Date | 绑定时间（格式：yyyy-MM-dd HH:mm:ss） |
| `todayDivideGold` | BigDecimal | 今日分成金币 |
| `todayDivideAmount` | BigDecimal | 今日分成佣金 |
| `totalDivideGold` | BigDecimal | 累积分成金币 |
| `totalDivideAmount` | BigDecimal | 累积分成佣金 |
| `divideType` | Integer | 分销分类（1-一级，2-二级） |
| `goldRatio` | BigDecimal | 金币分佣比例 |
| `rechargeRatio` | BigDecimal | 充值分佣比例 |
| `dataGeneration` | String | 数据生成标识 |

---

## 数据说明

- **一级成员**：直接绑定到当前会员的成员
- **二级成员**：绑定到一级成员的成员
- **总人数**：一级成员数 + 二级成员数
- **总利润**：所有成员产生的金币分佣 + 充值分佣总和
